# checkin-helper 自动签到助手 
部分功能来自 [@gxitm](https://github.com/gxitm/Autocheckin) 的项目。

使用说明：在config.json中加入自己的cookie即可

### 目前已实现

***网站：***
- 什么值得买 
- 联想社区签到
- V2EX
- 天翼云盘

***微信推送***

签到结果使用[SeverChan](http://sc.ftqq.com/3.version)推送至微信，将SCKEY添加到config.json的SEVERCHAN字段
